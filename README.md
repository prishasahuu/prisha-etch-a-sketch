# prisha-etch-a-sketch

I created an etch-a-sketch game online! This game has multiple features:
- Customizable Grid Size: Upon prompting the user, the game creates a customized grid size based on the size requested by the user.
- Eraser: Upon clicking the 'Eraser' button, the user can click on the squares they would like erased and erase it that way! (a modified, much cooler etch-a-sketch)
